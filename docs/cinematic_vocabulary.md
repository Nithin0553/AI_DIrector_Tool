# Cinematic Vocabulary

This document defines the allowed cinematic terms that the AI Director tool can use when generating beat scripts.

Using a controlled vocabulary ensures the system produces consistent and interpretable cinematic instructions.

---

# Shot Types

These describe the framing of the camera relative to the subject.

- wide_shot
- medium_shot
- medium_close_up
- close_up
- extreme_close_up
- over_shoulder
- two_shot
- reaction_shot

---

# Camera Angles

These describe the vertical angle of the camera.

- eye_level
- low_angle
- high_angle
- dutch_angle

---

# Camera Movement

These describe how the camera moves during the shot.

- static
- pan
- tilt
- dolly_in
- dolly_out
- push_in
- pull_out
- handheld

---

# Shot Transitions

These describe how one shot changes to the next.

- cut
- dissolve
- fade
- match_cut

---

# Character Emotions

These describe the emotional tone of a dialogue or action.

- neutral
- calm
- curious
- fear
- anger
- tension
- relief
- sadness
- shock

---

# Lighting Mood

These describe the lighting style of the scene.

- natural
- soft
- dramatic
- dark
- high_contrast

---

# Scene Types

These describe the narrative function of the scene.

- dialogue_scene
- confrontation_scene
- suspense_scene
- emotional_scene
- action_scene

---

# Notes

The AI Director must only select values from this vocabulary when generating beat scripts.

This controlled vocabulary ensures that the Unity engine can reliably interpret camera and scene instructions.
